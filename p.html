HTML

Tentang Preprosesor HTML

Preprosesor HTML dapat membuat penulisan HTML lebih ampuh atau nyaman. Misalnya, Markdown dirancang agar lebih mudah ditulis dan dibaca untuk dokumen teks, dan Anda dapat menulis perulangan di Pug.

Pelajari selengkapnya · Versi

Menambahkan Kelas

Di CodePen, apa pun yang Anda tulis di editor HTML akan masuk ke dalam <body>tag <head> di template HTML5 dasar . Jadi Anda tidak memiliki akses ke elemen yang lebih tinggi seperti <html>tag <body>. Jika Anda ingin menambahkan kelas di sana yang dapat memengaruhi seluruh dokumen, di sinilah tempatnya.

Tentang <head>

Di CodePen, apa pun yang Anda tulis di editor HTML akan masuk ke dalam <body>tag <head> di template HTML5 dasar . Jika Anda membutuhkan sesuatu di bagian <head></head> dokumen, letakkan kode tersebut di sini.

Tentang Preprosesor CSS

Preprosesor CSS membantu mempermudah penulisan CSS. Semuanya menawarkan hal-hal seperti variabel dan mixin untuk memberikan abstraksi yang nyaman.

Pelajari selengkapnya · Versi

Basis CSS

Tentang CSS Base

It's a common practice to apply CSS to a page that styles elements such that they are consistent across all browsers. We offer two of the most popular choices: normalize.css and a reset. Or, choose Neither and nothing will be applied.

Vendor Prefixing

About Vendor Prefixing

To get the best cross-browser support, it is a common practice to apply vendor prefixes to CSS properties and values that require them to work. For instance -webkit- or -moz-.

We offer two popular choices: Autoprefixer (which processes your CSS server-side) and -prefix-free (which applies prefixes via a script, client-side).
